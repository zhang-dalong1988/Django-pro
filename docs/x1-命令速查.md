# 命令速查

## Python

```bash
# 创建虚拟环境
$ python -m venv .venv
# 激活虚拟环境
$ source .venv/bin/activate
# 退出虚拟环境
$ deactivate
# 生成 requirements.txt
$ pip freeze > requirements.txt
# 安装依赖
$ pip install -r requirements.txt
```

## Git

```bash
$ git config --global user.name "Your Name"
$ git config --global user.email "yourname@email.com"
# 设置默认分支为 main
$ git config --global init.defaultBranch main
```

## Docker

```bash
# === Docker ===
# 使用当前目录下的 Dockerfile 构建镜像
$ docker build .

# === Docker Compose ===
# 使用当前目录下的 docker-compose.yml 启动容器
$ docker-compose up
# (分离模式)后台运行容器，不阻塞终端
$ docker-compose up -d

# 在容器中执行命令
$ docker-compose exec <service> <command>

# 查看容器日志
$ docker-compose logs

# 停止并删除容器
$ docker-compose down
```

## Django

```bash
# 安装 Django
$ pip install django

# 创建 Django 项目
# 最佳实践：手动创建项目目录，然后在项目目录下创建 Django 项目
# $ django-admin startproject _django_project .
$ django-admin startproject <project_name> <project_path>

# 迁移数据库
$ python manage.py migrate

# 启动开发服务器
$ python manage.py runserver

# 创建 Django 应用
$ python manage.py startapp <app_name>
```
