## 环境说明

Django 5 需要以下环境：

- Python 3.8+ -- 本地 3.13.5
- Django 5.2.4 -- 本地 5.2.4
- PostgreSQL 12+ -- 本地 14
  - psycopg2 >= 2.9.6 -- 本地 3.2.9
  - psycopg >= 3.1.8 -- 本地 3.2.9
